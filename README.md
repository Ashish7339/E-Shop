# 🛒 E-Commerce Website (Next.js)

A modern and responsive simple E-Commerce web application built using **Next.js**, where users can browse products, add items to cart, and manage their shopping experience seamlessly.

---

## 🚀 Features

* 🛍️ Product listing using external API
* 🔍 Dynamic product data fetching
* 🛒 Add to Cart functionality
* ➕➖ Increment / Decrement product quantity
* ❌ Remove items from cart
* 💾 Cart data stored in localStorage
* ⚡ Real-time cart updates (Total price & item count)
* 🎨 Responsive UI using Tailwind CSS

---

## 🧠 Tech Stack

* **Frontend:** Next.js (App Router)
* **State Management:** React Context API
* **Styling:** Tailwind CSS
* **Data Source:** External API
* **Storage:** localStorage
