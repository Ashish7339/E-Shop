## what is next js?
 - Next js is a react based framwork thats makes building  web application easir  and faster it add extra feature  on top react like 

 1. Server-side-scripting (SSR) -  page  load faster  and are better seo .
 2. File based routing - no need to manually set up routes, just create files.
 3. API Routes → you can build backend APIs inside the same project.


 ## Server-Side Scripting
 - “Server-side scripting means code runs on the server before sending the result to the browser. It handles logic like database operations, authentication, and API calls. Example: Node.js, PHP, Python (Django/Flask).”


 ## Client-Side Scripting
 - “Client-side scripting means code runs on the user’s browser, mainly for UI updates and interactions without reloading the page. Example: JavaScript, React, Angular.”
